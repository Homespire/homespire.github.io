Full Migration — Scope

Full migration means launching
six projects simultaneously.

Before a single Homespire account could be moved to NFM's environment, all six of these workstreams would need to be underway. Each requires its own dedicated resources, timeline, and leadership ownership — and several involve departments beyond IT.

This is not a technology migration. It is a program of work spanning compliance, legal, finance, operations, and IT across both organizations. Every project shown below must be resolved or actively in progress before any cutover can be attempted.

⚠️ This is a program, not a project. Each workstream requires its own project manager, dedicated staff allocation, and executive sponsorship. Critically, all timelines are contingent on NFM IT's availability and priorities — if the NFM IT Department is engaged on other initiatives, these estimates will extend accordingly. The full program also requires third-party migration tooling, and the sustained involvement of both IT departments, NFM Marketing, Legal, Compliance, and Finance. Hybrid Migration achieves meaningful integration in 6–8 weeks with none of this overhead.
